Гамма-функция $\Gamma(x) = \int_0^{+\infty} t^{x-1} e^{-t} dt$

__Гамма-функция непрерывна__ при $x > 0$

__Основное свойство гамма-функции__ $\Gamma(x+1) = x \Gamma(x)$

$\Gamma(x+1) = \int_0^{+\infty} t^x e^{-t} dt = -t^{x} e^{-t} |_0^{+\infty} + \int_0^{+\infty} x t^{x-1} e^{-t} dt = x \Gamma(x)$

`Gamma(x)=(Gamma(x+1))/x`

С помощью этого свойства можно продолжить функцию на отрицательнеую полуось и понять, что:

$\lim_{x\rightarrow 0+} \Gamma(x) = +\infty$

$\lim_{x\rightarrow +\infty} \Gamma(x) = +\infty$

__Производная гамма-функции__ $\Gamma'(x) = \int_0^{+\infty} t^{x-1}e^{-t}\ln(t)dt$

Ну доказываем там равномерную сходимость и тупо дифференцируем.

$\Gamma''(x)=\int_0^{+\infty} t^{x-1}e^{-t}\ln^2(t)dt > 0 \implies \Gamma'(x) \text{ возрастает}$
